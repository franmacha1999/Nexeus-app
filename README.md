# Nexeus - Plataforma de Gestión de Innovación y Tiempos

Plataforma web full-stack diseñada para la gestión interna de proyectos mediante tableros Kanban, control preciso de horas trabajadas y generación de informes semanales.

## 🚀 Tecnologías Utilizadas
- **Frontend:** React, Vite, Tailwind CSS, React Router, dnd-kit.
- **Backend:** Python, FastAPI, SQLAlchemy, PostgreSQL.
- **Despliegue:** Vercel (Frontend) & Render (Backend con base de datos en PostgreSQL).

## ✨ Características Principales
1. **Autenticación Segura:** Sistema de registro e inicio de sesión basado en tokens JWT.
2. **Tablero Kanban Dinámico:** Arrastrar y soltar iniciativas entre columnas (`Backlog`, `En Progreso`, `Revisión`, `Listo`) con persistencia en base de datos.
3. **Control de Horas (Time Tracker):** Temporizadores integrados en tarjetas y hojas de tiempo semanales personales.
4. **Business Intelligence:** Informes semanales de rendimiento y exportación de datos en formato CSV.
5. **Filtros Avanzados:** Búsqueda en tiempo real por texto y filtrado por niveles de prioridad.
